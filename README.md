# dc-SQUID potential sweep and viewer

This repository contains the code used to generate dc-SQUID potential plots from the PhD thesis of Achintya Paradkar for generating and inspecting 3D potential landscapes of an asymmetric dc-SQUID.

## Files

- `dc_squid_potential_sweep.py`  
  Generates potential landscapes and tracked stationary states over parameter sweeps.

- `dc_squid_potential_viewer.py`  
  Interactive viewer for browsing generated PNG outputs across parameter values.

## Purpose

This repository contains the code used to generate dc-SQUID potential plots for the PhD thesis of Achintya Paradkar.
